2020年欲拒还迎

市场分析

即将过去的2019年是很不错的年份。

大盘行情先抑后扬，从2018底调整的最低处绝地求生，一路反弹，在5月和7月两次摸高，然而受制于贸易不确定性未果，之后稍作徘徊，终于在年底晴空万里，气势如虹，站上了3200点高位，全年标普大盘飙升3成，是2013年以来最佳年度表现。

那么我们怎么看即将到来的2020年？

2020年的功课

在狂欢后坠落和派对才开始中我们捕捉到了四季度的乐观情绪，然后我们又在 排雷中排除了一些障碍，提升了对2020年的期待。那么，是否已经万事大吉？

投资者会想当然地认为短期动能可以延续。然而，从历史经验来看，这不可取。

人们总是忽视可能发生的各种意外事件。其实，意外事件的必然出现，才是唯一确定性。

管理风险而非盈利。风险可以把握，盈利则靠市场。所以，我们有必要从风险角度出发，做好2020年的功课。我列出了目前能想到的风险：

  • 美股企业盈利未能恢复势头，甚至下滑；
  • 股市过高的估值终于坍塌；
  • Fed再次逆转政策，重新加息或者削减负债表；
  • 公司债出现信用危机，引发连锁反应；
  • 出现和2016年类似的震惊的选举结果；
  • 第一阶段协议履行出现问题，贸易冲突再起；
  • 全球其他地方经济恶化，比如强国；
  • 地缘冲突爆发，中东和东北亚是热点地区；

前面四条是经济风险，后面四条是政治风险。

高估值

上面清单中，美股高估值是明显的灰犀牛。

高估值如同蓄满水的高坝，不知道那一道裂缝将导致堤坝崩溃，这让我时刻悬着心。

美股今年涨了29%，然而上市公司利润只增长了0.5%。利润增长通常是股价持续走高可靠的动力，所以当股指攀新高而利润增长落后时，我们会担心市场情绪骤然反转，就像2000年互联网泡沫破裂那样。

市盈率PE在30左右，估值在攀升：

不仅PE，市销率PS和企业乘数 EV/EBITDA 都处于历史高位。

这预示着后面10年股市的回报将显著下滑。从历史统计上看，未来数年回报将是比较低的个位数字。

上市公司的盈利和估值有密切关系，如果公司盈利大有改善，估值就会缓解。如果公司盈利没有改善甚至恶化，高估值将难以维持。

很多投资者却没有看到这点，还在将钱不断投入到风险资产中，这是泡沫的典型征兆。尽管技术面显示未来回报仍然积极，但是仅就估值而言，防御是审慎可取的做法。

2015年后，在两次大盘牛熊之辩的关键时刻，欧洲央行和Fed都果断出手，逆转了势头。可以说2015年后的牛市，是打强心针续命到如今的。


公司债

美国公司债和垃圾债，我们在下一场衰退怎么来？ 中讨论过。

简单补充一下，不到10年BBB级公司债就从6860亿膨胀到2.5万亿，当前规模堪比2008年的次级贷。公司债和GDP的比例已经攀升至历史高位。

未来5年，一半的公司债将到期。BBB级公司债离垃圾债只有一线之隔。如果有一家稍有规模的公司出现偿付危机，这就像当前刺穿次贷泡沫的雷曼兄弟一样，引发连锁反应。或者还有一种更大可能，如果经济景气下滑，导致大量BBB级债券降级为垃圾债，将引发基金大规模抛售。这很恐怖，因为没人接得了盘。

剧变中的世界

2019年也许是冷战以后全球化黄金时代的落日余晖，往后的世界将面目全非。这几十年的出奇繁荣，在历史上未曾有过，以后也不会再有。

我们交易的虽然是美股，但是美股与强国息息相关，2018年以来这种关联不断得到市场证实。

强国经济在未来将面临诸多挑战：

  • 人口增长停滞，结构老化；
  • 内部的高压锅气候导致创新型经济无法生长；
  • 外部压力加剧；
  • 系统原因不利于民营经济，拖累就业和出口；
  • 比美国更严重的公司债务危机；

解决这些问题要有柔软的身姿，果决的手段，特别要打破固有藩篱，与人民休戚与共，谋百世而不谋一时。当然，看客只能做一做键盘侠，当家才知油米贵。

世界其他地方的危机也会有，但是我感觉大洋这边的风险才最要紧。

概率计算

决策，其实是对于不同结果的可能性分配概率，然后汇总概率得到最终结果罢了。

过去120年，其中有83年股市是上涨的，所有有83/120 = 69%的概率，明年股市上涨，这是基础概率。

然后，在这个基础上按照风险和利好的多少，进行概率的调整。

比如，美股大选年一般行情表现较好，所以上涨概率+10%。然而美股估值高出平均值两倍标准差，大幅度上涨概率-20%。如果Fed能适应政治需要维持宽松-我相信会的-则Fed的宽松政策将+10%的上涨概率。如此一一枚举下去。

有点拍脑袋，毕竟不专业，没有数据积累，不能量化。但是主观估计的正负误差也可以互相抵消，所以想一遍终归有好处，至少给了自己信心。

如此这样算下来，2020年很可能还是一个正回报的年份，但是不要期待过高，毕竟周期走到了尾声。我斗胆估计3300点是目标低位，也许可以高看到3400点。

这不是预测，只是基于已知的事实进行概率计算。

我们心里仍然明白，预测不可靠的也不可取。在漫漫长路中，我们只是努力搞明白自己的方位。

欲拒还迎

大盘会继续攀升，但是如果在当前情况下还全仓股票，就忽视了泡沫的危险。

此时，谨慎地持有几家精心挑选的防御性股票，或者使用期权和债券进行保护和对冲是可取的。

一方面保护已有的收益，一方面限制下行的风险。

2020年确实很大可能有正回报，但是也有诸多风险，这不是看空，而是提醒大家小心应对。

  • 如果错了，资产组合只是表现稍微弱于大盘。
  • 如果对了，由于采取防守姿态，资产组合将免于大破，优于大盘。

过去多年中牛市获利只做完一半的功课，另一半功课是要在下面几年保护好辛苦钱。

总之，以欲拒还迎，欲说还休的状态面向2020年。

也预祝大家2020年云帆高挂，风生水起，财路通畅，直济沧海！
