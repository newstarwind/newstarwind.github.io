麦克斯韦小妖#

生命的复杂度也影响着遗传漂变。

我们知道，演化的动力是自然选择。自然选择为什么会推动生命变得越来越复杂，这是一个深刻的问题。我试图从一个本质的视角出发给出简单的解释。

让我们回到最初的开端 — 热力学第二定理，也就是熵增定律上来。

热力学第二定律是所有物理定律中唯一能够区分过去和现在的定律，它定义了时间的方向。它也是一个不能被任何其他定律所推导的定律，它纯属经验所得。

第二定律这么独特，导致一些物理学家对它不以为然，其中就包括大物理学家麦克斯韦。麦克提出了麦克斯韦方程组，完美确立了电磁领域的数学规律。在1871年的《论热能》一书中，麦克斯韦提出一个问题来诘难第二定律，他举了一个例子：

一个箱子中间被一块垂直的板子隔开，板子上有一个活门，活门由一个小妖怪看守。小妖怪的任务是，观察要穿越小门的气体分子。它看到左边来的分子，如果速度慢就放行，如果速度快就不放行。对于右边来的气体分子，如果速度快就放行，如果速度慢就不放行。这样，箱子左边的气体分子的速度就会保持在高位，而箱子右边的气体分子就会保持在低位。一段时间后，整个箱子中的气体的熵值就变低了。这是不是违背了热力学第二定律呢？

按照第二定律，要降低封闭系统的熵值，就必须做功。但是，此处的小妖怪只是负责开门关门，这种动作的功率可以小到忽略不计，为此麦克斯韦还专门提供了活门的可行设计。

麦克斯韦在书中总结说：

热系统左边变得更热，右边变得更冷，然而却没有做功，只有一个眼光犀利、手脚麻利的智能生物在工作。

没有做功而熵值降低，多么美好啊。

麦克斯韦的小妖难到了不少当时的杰出头脑。许多物理学家都坚持说热力学第二定律没错，猫腻出在那个小妖身上，但是他们又说不出个所以然来。

直到1929年，这个问题才被解决。匈牙利物理学家西拉德提出，做功的是小妖的智能，更精确地说，是通过测量获取信息的行为。西拉德是第一个将熵与信息联系起来的人。

小妖的测量 - 回答是与否的过程 - 是需要消耗能量的，这些能量就是熵增，它与箱子中气体的熵减可以互相抵消。这样，箱子、气体、小妖组成的整个系统仍然符合第二定律。

到了1960年前后，物理学家兰道又发现，在小妖的判断过程中，既要存储信息，也要擦除信息，而擦除信息也是需要增加熵值的。

这样，经过了近一百年，兴风作浪的麦克斯韦小妖终于被平定了。

那么，我们此时为什么要谈麦克斯韦小妖？

因为，在演化中无数变异的个体中挑选可以存活的少数，自然选择同样是在进行测量和判断，它就是麦克斯韦小妖。自然选择会说，这只生物奔跑快，可以去左边。那只生物的视力不好，反应不够快，呆在右边吧。左边的延续，右边的绝嗣。

自然选择的做功必然导致生命这个系统，熵值降低，趋于复杂而有序。

为了保持负熵，在内部进行新陈代谢，生命体必须要有边界，将自己与环境区分开来。

让我们看一看生命体内的各种 膜 吧。细胞膜、羊膜、表皮、胎盘、基底膜等等。这些膜的一个共性，就是选择性透过。一些分子，比如水、氧气和葡萄糖可以透过，而另一些分子，比如病毒或者异种蛋白质，则不能透过。通过这些选择，生命体维持了自己的秩序，保持了负熵。这些膜，就是生命体内部的麦克斯韦小妖的各种化身，也是边界。

演化中的自然选择当然也消耗能量，它消耗的是一代代被淘汰的无数生物个体，以及随着这些个体的毁灭而耗散的能量。当然，这些能量不会被平白浪费，而是被其他活着的生物再次利用，作为食物或者有机质来源。换句话说，自然选择以无数生物个体的生命为代价，推动着整个生命之树向着有序复杂的方向生长。

约翰霍兰德，也是遗传算法之父，的经典之作 《隐秩序：适应性造就复杂性》就专门阐述这个主题。

自然选择就是我们前述的 宇宙的创造性 在生命上的直接体现。

由于生物对于环境的适应性带来其复杂性，那么复杂性的来源就从生物本身转移到了环境上来。为什么环境是复杂的？这里面有两个主要原因。

其一，物理规律的内在本质 导致了生物的生活环境的差异。地球上有海洋、大陆与岛屿，有寒带、温带和热带，有季风、洋流、冰川和各种气候，有沙漠、草原、荒原和森林等等地形。这些环境的差异都会影响生物种群的生活，驱动它们朝向不同的适应性而演化。

其二，生物适应环境则趋向多样性。为了适应具体的生存环境，生物会强化某些能力，而牺牲一些其他的能力，这种变化称为特异化。比如洞穴生物就不需要视力，而强化了嗅觉和听力。天上的飞禽则无需嗅觉，强化了视力。人类先祖作为一种在非洲草原上的直立生物，开阔的视野特别重要，所以人类的头脑特别强化了视觉处理，而弱化了同为兽类所擅长的嗅觉。

自然选择驱使个体之间的差异扩大，导致生物多样性。多个物种的竞争，加上环境本身的变化，环境中生态位的分化，这些复杂性将驱使物种继续朝向新的特异性发展，产生更多物种。多样性导致复杂性，复杂性导致多样性，这是无止境的互相强化的过程。

如果继续沿用小妖这个类比，我们可以认为，不同的环境就像不同的盒子，每个盒子里面都有一个自然选择扮演的小妖在做功。地球上，无数差异的环境中，都有无数个自然选择的程序在运行，它们就是无数个看门的小妖。它们的共同作用下，地球上的物种体现出丰富的多样性，欣欣向荣，枝繁叶茂。

复杂性的光谱#

复杂性存在于复杂系统 之中。

目前科学家们对于复杂性到底如何测量尚无定论。我认为复杂性可以用这两样东西来衡量：

用逻辑描述的规则 — 可以类比为计算机伪代码 — 的长短，以及其中包含的随机性。

形象地说，我们可以将最单一的规则作为左侧顶点，最大的随机性作为右侧顶点，画一条直线连接二者。这条线段的两端，最大的随机性和最单一的规则性都是复杂程度最低的，二者中间的区域才是复杂程度最高的，这里既有规则又有随机性，时刻处于动荡之中，这里就是大家通常所说的 混沌的边缘，也是演化出新的自组织系统的发源地。

在创世之初的奇点时刻，宇宙是无差别无时空的纯粹能量，此时是最简单的规则，是线段的左顶点。而未来的热寂状态，则是最大的随机性，没有任何规则，所有的物质都分崩离析，是线段的右顶点。熵增，就是从线段的左端走向右端。

在这个过程中，通常会包含两个阶段：

  • 熵增而复杂性增加，系统的信息含量不断上升，系统处于孕育发展，欣欣向荣的阶段。
  • 熵增而复杂性下降，系统的信息含量不断下降，系统处于趋向败坏，崩解失序的阶段。

上述这个过程不仅仅发生在宇宙层级上，也发生在封闭的复杂系统中。生命就是这样的复杂系统。

熵增是始终的，但是系统的复杂性却先累加而后降解。所以，除了熵和能量这两个根本变量之外，还有一个指标来衡量系统的复杂性，我们通常称之为 信息。对于生命而言，信息是与熵、能量等量齐观的根本属性。

在演化过程中，生物本身的信息量，是趋于增大的。但是也有一些例外情况，比如某些寄生虫或者病毒，由于其生存策略的专一，一招鲜吃遍天，它们会放弃掉其他任何与寄生无关的机能，而趋于简单。这种演化的代价是它们失去了通用的适应能力，最终和宿主绑定在一起。

大体上，生命的复杂性至少体现在三个方向上。

其一是广度。生命门类繁多，生命具有多样性。

其二是深度。生物个体本身的构造趋于复杂，体现为生物的分层结构以及同一个层面上出现许多组成部分。比如，人是由很多个器官构成的，器官是由无数细胞构成的，细胞是由很多细胞器细胞质构成的，细胞器实际上是无数个蛋白质和酶等大分子构成的分子工厂。每一个层面都有其独特的运行规律，层面之间是松耦合的、嵌套的。整个生物体表现出了逻辑深度上的复杂性。

其三是 自举 。这个概念不太好简单地正面解释，我打个比方：如果科学家某天得到了恐龙的完整DNA，是否可以像《侏罗纪公园》中那样复制出活生生的恐龙？电影中可以，实际上却完全不可以，原因很简单，因为科学家们得到的不是活的恐龙蛋，而是DNA。

DNA是程序，需要对应的解释器才能运行。恐龙蛋由细胞组成，可以自己生长发育，是完全的可执行程序。DNA却不行，它包含数据，但是它如果不处于一个活的细胞的细胞核中，则没法开启自我复制过程。这就像任何代码的运行都离不开代码编译器解释器一样。

可不可以根据恐龙的DNA造出这样一个细胞？估计是不行的，因为DNA的解释器也编码在DNA本身之中，下一代的DNA总是依赖上一代的解释器来生成的，这是一环套一环的递进关系。

那么，最初恐龙是怎么来的呢？是先有鸡还是先有蛋？

我们可以从演化的角度来看这个问题。鸡与蛋的关系可以不断前推，最终找到这样一个依赖链条：

最早的生命 → 最早的动物 → 有性生殖的动物 → 卵生动物 → 较复杂的卵生动物（两栖动物、爬行动物）→ 鸟类。

每个物种都在世代更替中繁衍，又在繁衍中变异。如果一个物种中某个体的变异大到无法和其他同类个体进行有性生殖时，新物种就此产生了。从这个角度理解，就会发现鸡与蛋无非是一个循环中的两个不同的要素，少了其中任何一个这个循环都无法继续推进，因此鸡与蛋不存在孰先孰后的问题。

当几亿年前某个有性生殖的生物一次偶然地将受精卵排出了体外的池塘中，而那个卵碰巧没有死，还长成了的时候，逻辑意义上的鸡和蛋就同时有了。演化史中，该生物应该是一种原始的水中物种。

因此，生命的复杂性也是因为演化路途中环环相扣的变异带来的，是 路径依赖 的，是偶然而不可复现的。我们现在看到的全部物种都来自35亿年前那个最初的细胞。它就是盖亚，所有生命的母亲。它自己给自己搭梯子，沿着复杂性的金字塔攀登。它自己给自己先后升级了细胞核、线粒体、叶绿体、有氧呼吸、有性繁殖、眼睛、肌肉、神经元、脊柱、大脑等等装备，最后成为你我这样的生物，这个过程就是自举。

在自然选择这名法官的威严注视之下，自举过程命运多舛，筚路蓝缕，也曾偶尔峰回路转。所以，物种都是独特的，并没有金科玉律可以决定那个物种一定会出现，那个物种一定会繁盛。

如果地球再重新来过，人类几乎肯定不会再出现了。

在 《奇妙的生命：布尔吉斯页岩中的生命故事》中，斯蒂芬·杰·古尔德 说，假如我们“将生命倒带”，并一遍又一遍地播放它，那么无论再把演化之轮推动多少次，产生出我们的可能性都会无穷小。

复杂性很深奥，我们将一再回到这个问题上来。

复杂性影响着演化#

在演化过程中，一些物种趋向于越来越复杂，它们在身体构造、体型、行为方式、群体结构等等方面莫不如此。随着进化创造出越来越复杂、越来越大型的生物体，遗传漂变的作用也缓慢稳定地增强。

前面说过， X病毒的基因共有一万个字符。每次突变影响一个字符。我们假设遗传漂变带来的震动幅度是基因的 1%，则其变化的跨度是 $100 * 100 * 0.01 = 100$ 个字符，它比突变大了100倍，这就是为什么遗传漂变可以帮助X病毒跨过山谷的原因。

病毒一般大小在几十到几百纳米，而细菌的大小在几微米之间，二者的大小相差大约百万倍。

假设有一种 B 细菌，它个头比 X 病毒大多了，结构也复杂不少。它的基因也多了不少，有1000个基因，每个基因有1000个字符。这样，在相同的遗传漂变的震幅下，B细菌的变化跨度是 $1000 * 1000 * 0.01 = 10000$ 个字符，这个跨度是 X 病毒的遗传漂变的一百倍。

较大生物体的基因组通常也更为复杂，包含更多重复的、有缺陷的、不活跃的基因。比如人类有两万多个基因，一共30亿个字符，其中只有大约 2% 的字符是明确控制蛋白质生成的，其他的字符有的起到控制作用，有的仅仅是用途不明的 “垃圾DNA”。同样振幅的遗传漂变，作用在人类身上的变化跨度是 $30亿 * 30亿 * 0.01 = 3千万$ ，这个跨度是 B 病菌的三千倍，是 X 病毒的三十万倍。

另一方面，生物体个头越大，它需要的生存空间就越大，这意味着较大的生物通常生活在规模较小的种群中。

细菌大约是几亿个体生活在一起，昆虫大约是几千万个体生活在一起，而脊椎动物和树木的群落规模一般在一万以下，那些顶级的掠食者，比如狮子老虎之类的，大约需要几十平方公里才能养活一只。一克粪便中病毒就有 $10^9$ 个，可以说病毒的群落规模几乎是不计其数。

所以，总的趋势是生物体越大其种群规模就越小。与较小的生物体相比，在较大生物体的较小种群中，遗传漂变的影响也更大，同时自然选择的功效则远远不如。这样，许多有害的等位基因在细菌种群中很快就会被消灭，而在大型动植物种群中却能长期持续存在，就仿佛自然选择对它们视而不见一样。

可以这样看：

微生物的基因库就像是苦行僧的小屋，里面没有一件多余之物。
大型生物的基因库则像是一间业余发明家的地下车库，里面存放着各种零件工具，陈年杂货，甚至只做了一半的物件。

这一团乱糟糟的家什正是下一次创新的伏笔。遗传漂变就是那个业余折腾者，在适当的自然选择作用下，它能够持续地在基因库上东敲西打，修修补补。基因库中那一堆老物件也可以随时用新的方式混搭起来，发挥作用。

总之，基因漂变将演化从自然选择的鼠目寸光中暂时解放出来，使种群不再一味地埋头爬坡，此时种群才有可能在崎岖的地形中探索更大的景观区域，进而演化出更精彩的形式。

演化是在自然选择与遗传漂变之间的均衡，是戴着镣铐的舞蹈，它时而奔放，时而踌躇，边走边唱，踯躅前行。

复制性的层级#

在我们肉眼所及的世界中，复杂系统几乎都有着多级分层结构，像是洋葱那样一层叠一层。

1962年 复杂理论的开创者西蒙 发表了一篇宏文《复杂性的结构》，西蒙认为，复杂系统最重要的共性就是层级性和不可分解性。

  • 层级性：复杂系统是多个层级的累加而成。比如，身体由器官组成，器官又是由细胞组成，细胞中又含有细胞器。
  • 不可分解性：在层级中，组分内部的紧密作用比组分之间要强得多。例如，细胞内部的新陈代谢就比细胞之间的相互作用要复杂紧密得多。这就像，多个砖块叠加成墙，墙壁上开门框与窗户，再进一步搭成房屋。砖块内部是紧实坚固的，而砖块之间是用水泥石灰粘在一起的。

自然界中的复杂结构之所以能以层级的方式被组织起来，那是因为，层级结构提供了最可行的扩展方式。

简单地说，自然界中的复杂系统都不是一开始就复杂的，它们基本上都是由简单系统生长而来。比如，一个基本的独立分子 — 氨基酸，如果它成功地在演化中存活下来了，那么在后续演化中，它就具备了一种可能的扩展方式，就是以多个同类的聚集 — 就像水洼里聚成一团的蛙卵一样 — 形成一个组合，然后再在这个层面上进行分工与协作，达成更复杂的能力，进而逐渐成为一个整体。

多个不同的氨基酸分子聚成一团，成为蛋白质，蛋白质就可以完成某项独特的功能。有的蛋白质成为隔绝内外的渗透膜，有的成为可以产出能量的反应器。多个蛋白质大分子再继续组合，可以成为更大的整体，比如细胞的一部分，如此不断向上扩展。这之中，我们前提到的算子 — RNA 和 DNA，起到了核心的组织协调作用。

在软件领域，操作系统、数据库、分布式系统以及大型软件的架构，都离不开分层结构。互联网就是基于7层嵌套的网络通讯协议而搭建的。

生命作为复杂系统，具备细胞、组织、器官、个体等不同的层面。每个层面都有自己的结构和功能，层面之间有物质、能量和信息的交换。这些层面各有其能力表现，它们一层叠一层，涌现出了超越单一层次的特点，例如分层进化、自我组织、自我修复等，从而产生适应性。

人类社会也是一种复杂系统，它包括个人、家庭、社群、国家、国际等不同的层面，每个层次都有其结构与功能。这些分层结构使得人类社会具备弹性，涌现出超越任一单个层面的特性，例如文化、制度、文明等等。

在物种演化中，由于自然选择的驱动，生物的变化必然同时发生在生物体的所有层面上，这些变化合在一起，促进生物整体适应性的提升。

例如，人在世代繁衍中，基因在分子层面演化，提供不同蛋白质的合成。而人体的细胞也在更高层面同时发生着演化。以最重要的能量供应机制为例，我们发现人体细胞内部有三种不同的能量机制，其演化发生的时间和源头各不相同，它们一起作用在细胞中，提供不同场合的能量供应：

  • 糖酵解：最早的单细胞生物，如原核生物和原始真核生物，主要依靠无氧的方式来获取能量，即通过糖酵解将葡萄糖分解为乳酸或乙醇，释放少量的ATP。它简单快速却效率低下，只能利用葡萄糖的一小部分能量，而且会产生有毒的乳酸或乙醇。它来自于生物最初生活的缺氧环境。

  • 有氧呼吸：随着地球大气中氧气的增加，一些单细胞生物开始利用有氧的方式来获取能量，即通过有氧系统将葡萄糖完全氧化为二氧化碳和水，释放大量的ATP。这种机制复杂持久，而且效率高，可以利用葡萄糖的全部能量，并且不会产生有毒的副产物。它来自蓝藻改造了地球大气之后，生物适应了富含氧气的环境。

  • ATP-CP系统：在有氧系统的基础上，一些多细胞生物进一步发展出了ATP-CP系统。它可以在无氧条件下快速合成ATP（三磷酸腺苷），为肌肉收缩提供能量。它的特点是功率大、时间短，是短跑、跳跃、举重等爆发力运动的主要供能方式。它非常快速，但效率极低，而且很快就会耗尽。

这三种不同的能量机制都编码在人类DNA中，人体同时运用它们。

除了细胞这一层之外，人体在器官层面上也同时发生着各种演化。不同的个人，有着明显差异的血型、眼睛、皮肤、体型等等差异。在人群这个层面上，演化也同时发生着。不同的人群有不同的风俗，比如方言、饮食、婚丧等等习俗都不太一样。如此等等，我就不赘叙了。

生命是典型的复杂系统，所以我们可以说：

演化不仅发生在复杂系统的某一个层面上。在其任意层面上，都同时发生着演化。

但是，在所有这些层面上的演化结果，都会下降到 DNA 层级，在那里进行数据的存储和组合。DNA 可以持久化 — 也就是存储和表现 — 它所构造的多个层面的演化。

比如，生命从细胞器，细胞，组织，器官到个体的多个层面的表现功能的演化都在同时进行着，撇开各个层级之间的互相作用不谈，最后这些层面各自的演化结果都将作为数据存储在 DNA 中。DNA 字符的排列组合的改变，将可以影响所有这些层级，导致它们表现功能的差异。

在软件领域中有一个很常见的 MVC模式，它将构成软件应用的用户界面，商业逻辑和数据模型分离为三个从上到下的粗略层级。一般而言，不管是用户界面和商业逻辑如何改变，都不应该影响数据模型。但是如果数据模型发生变化，将肯定影响用户界面和商业逻辑。

在这个视图中，DNA 可以类比为数据模型，但实际上，DNA 远胜数据，它编码了无数的开关，可以根据环境的需要进行调整，DNA 也同时决定着物种个体的功能表现。
